# Subdomínios

- Partes da aplicação
- Devem ser independentes
- Existem 3 tipos:
  - Core: o que da dinheiro
  - Supporting: Dá suporte para o core funcionar
  - Generic: Precisamos deles, mas não tão importantes. Podem ser terceirizados talvez.

## Exemplos

### Core

- Compra
- Catálogo
- Pagamento
- Entrega
- Faturamento

### Supporting

- Estoque

### Generic

- Notificação ao cliente
- Promoções
- Chat
